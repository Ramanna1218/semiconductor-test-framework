# Automated Test Framework for Simulated Semiconductor Equipment

A comprehensive C# .NET-based test automation framework for validating semiconductor equipment through simulation.

## Features

 **19 Test Cases** across 5 domains
 **Multi-threaded Execution** (~30% performance improvement)
 **Advanced Logging System** with file and console output
 **JSON/CSV Result Export** for analysis
 **Hardware Simulation** with realistic behavior

## Quick Start

### Requirements
- Visual Studio Community 2022 or later
- .NET 6.0 SDK or higher
- Windows operating system

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Ramanna1218/semiconductor-test-framework.git
cd semiconductor-test-framework
```

2. **Open in Visual Studio:**
   - Launch Visual Studio Community
   - File → Open → Project/Solution
   - Select `SemiconductorTestFramework.sln`

3. **Build the solution:**
   - Press `Ctrl + Shift + B`
   - Should show "Build succeeded"

4. **Run the tests:**
   - Press `Ctrl + F5`
   - Console window opens showing test execution
   - Results display with pass/fail statistics
